# Wordspace

> A personal constellation of book excerpts, thoughts, and reflections.

Wordspace renders the passages you've highlighted from books as **stars in a candlelit room**. Stars near each other share semantic territory. Click one to read the excerpt, your thought, Claude's reflection, and what else in your library it's pulling on.

It's a sibling project to [Mindspace](https://github.com/amikovid/mindspace) — same 3D engine, different paint, different purpose. Mindspace is collective wisdom; Wordspace is private reading.

## Status

**Early.** The constellation renders with placeholder excerpts. The capture pipeline (Claude.ai → Neon DB) and analysis layer (themes, tensions, weekly digest) are designed but not yet wired up.

## Setup

```bash
npm install
cp .env.example .env
# edit .env: DATABASE_URL, OPENAI_API_KEY (ANTHROPIC_API_KEY can wait)
npm run dev
```

Open the URL Vite prints (typically `http://localhost:5173`).

### Run the schema against Neon

```bash
psql "$DATABASE_URL" -f scripts/schema.sql
```

…or paste `scripts/schema.sql` into Neon's web SQL editor.

### Regenerate the 3D positions

Edit `src/data/excerpts-seed.json`, then:

```bash
npm run generate-embeddings
```

This re-embeds, re-projects with PCA, recomputes nearest neighbors, and writes `src/data/excerpts-processed.json` (what the frontend reads).

## What's where

```
src/
  App.jsx              ← top-level state, ties everything together
  components/
    Scene.jsx          ← R3F Canvas, lighting, bloom
    StarField.jsx      ← star layout + connection lines
    Star.jsx           ← single star, candle flicker, hover/select states
    DetailPanel.jsx    ← excerpt + thought + reflection + related
    LandingScreen.jsx  ← entry overlay
    SearchBar.jsx      ← search with serendipity-first UX
    AudioController.jsx← optional ambient (off by default)
    LoadingScreen.jsx
  data/
    excerpts-seed.json       ← human-edited source of excerpts
    excerpts-processed.json  ← generated by scripts/generateEmbeddings.js
  styles/index.css     ← Tailwind + warm palette CSS vars

scripts/
  schema.sql                ← Neon Postgres + pgvector schema
  generateEmbeddings.js     ← offline embedding + PCA pipeline
```

## Roadmap

1. **Capture pipeline** — Claude.ai Project receives an excerpt + your thought, calls a write endpoint that inserts into Neon and triggers re-embedding.
2. **Analysis layer** — API endpoints for `/themes`, `/tensions`, `/digest`, `/deepen`, `/ask`. Some manual, some scheduled (weekly digest).
3. **Analysis drawer** — slide-out panel in the UI showing reflections, tensions, open questions.
4. **Book pages** — auto-fetch covers from Open Library; cluster excerpts by book.
5. **Theme labels** — faint floating labels at constellation centers, maintained by Claude.
6. **Reading mode** — expand an excerpt into a centered, well-typeset column.
7. **Time slider** — see how the library filled in over time.

## Stack

| Layer | Tool |
|---|---|
| 3D rendering | Three.js + @react-three/fiber |
| Post-processing | @react-three/postprocessing (Bloom) |
| Audio | Tone.js (FM pad + pink-noise fireplace bed) |
| UI animation | Framer Motion |
| Styling | Tailwind + custom CSS |
| Embeddings | OpenAI `text-embedding-3-small` |
| Reduction | PCA (`ml-pca`) → 3D |
| Storage | Neon Postgres + pgvector |
| Analysis | Claude (`claude-opus-4-7` or `claude-sonnet-4-6`) |
| Deploy | Vercel |

## Visual identity

Candlelit study, not deep space. Deep umber background, amber/gold stars with subtle flicker, copper-gold connection lines, parchment-tinted glass panels, EB Garamond for excerpt bodies.
